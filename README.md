# Kotlin Android App
Aplikasi Android berbasis Kotlin, menggunakan SQLite sebagai local database, dan dapat di-build menjadi APK Release.

# Tech Stack
Kotlin (Android)

SQLite (Room / SQLiteOpenHelper)

Android Studio (Gradle)

# Clone Project
```sh
git clone https://github.com/nagiazoe/Ujian-.git
cd project-name
```
# Database: SQLite
Project ini menggunakan SQLite yang sudah tersedia di Android 

# Menjalankan Project di Android Studio
1. Buka Android Studio

2. Pilih Open Project

2. Arahkan ke folder project

3. Tunggu proses Gradle Sync

4. Pilih device:

5. Emulator Android

6. HP Android dengan USB Debugging

7. Klik tombol Run 

# compile apk realese
```sh
./gradlew assembleRelease
```
# project apk realese berada:
app/build/outputs/apk/release/app-release.apk
